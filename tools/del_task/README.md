### del_task — 删除定时任务
- **用途**：当你需要删除一个已添加的定时任务时，使用此工具。它会从 `tasks.json` 中移除指定ID的任务，并自动将后续任务的ID重新排列为连续数字（例如删除ID=2后，原ID=3变为2，原ID=4变为3，依此类推）。
- **输入**：
  ```json
  {
    "id": 2
  }
  - `id`：整数，要删除的任务ID。
- **输出**：成功时返回确认信息，例如 `"已删除ID 2 的任务，并重新对齐任务ID（原ID大于2的任务ID均已减1）。"`。如果任务文件不存在或ID不存在，返回相应错误信息。
- **注意事项**：
  - 任务文件位于项目根目录的 `tasks.json`，格式为 `{"ID": ["内容", "时间"]}`。
  - 删除后，所有剩余任务的ID会重新从1开始连续编号。
  - 此工具与 `add_task` 配套使用，保证任务ID的连续性和唯一性。
