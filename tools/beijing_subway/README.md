### beijing_subway — 北京地铁换乘助手
- **用途**：根据出发时间，规划北京地铁从起点站到终点站的最优换乘路线（按时间最短），自动考虑早晚高峰速度差异和换乘时间惩罚。
- **输入**：
  - `start`：起点站名（如 `"西直门"`）
  - `end`：终点站名（如 `"北京南站"`）
  - `time`：可选，出发时间（距离当天 0:00 的分钟数，如 540 表示 9:00）。不提供则使用当前系统时间。
- **输出**：返回文字描述的换乘方案，包括乘坐线路、换乘站点、全程耗时（分钟）。
- **注意事项**：
  - 需要 `geopy` 库（用于计算站点间直线距离）。若未安装，运行 `pip install geopy`。
  - 数据文件 `station.json` 和 `graph.txt` 需与 `tool.py` 放在同一目录下。
  - 高峰期定义：早高峰 7:00-9:00（420-540 分钟），晚高峰 17:00-19:00（1020-1140 分钟）。高峰期地铁行驶速度较慢（35km/h），换乘惩罚增加（5分钟），平峰期速度40km/h，换乘惩罚2分钟。
  - 换乘惩罚因子可修改 `tool.py` 中的 `PEAK_PENALTY` 和 `OFF_PEAK_PENALTY` 调整。
  - 仅支持北京地铁线路，站点数据基于最新版本（若有线路调整需更新数据文件）。
